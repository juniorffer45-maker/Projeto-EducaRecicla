## Sobre o Projeto:
O EducaRecicla visa resolver dois grandes problemas: a dificuldade financeira de muitas famílias em comprar material escolar e o desperdício de itens que ainda podem ser usados. 
Sabemos que o preço dos materiais muitas vezes impede que alunos de baixa renda continuem estudando. Para mudar isso, nossa plataforma funciona como uma rede de economia circular. Nós conectamos quem quer doar materiais a quem precisa deles na mesma região. Assim, itens em bom estado voltam para as salas de aula em vez de irem para o lixo.
Para garantir segurança e facilidade, utilizamos escolas e bibliotecas públicas como pontos de entrega e retirada. Com isso, o projeto alcança três objetivos principais:
- Ajuda as famílias a economizarem e manterem seus filhos na escola.
- Protege o meio ambiente, evitando o descarte desnecessário.
- Une a comunidade, trazendo as famílias para dentro do ambiente escolar de forma confiável e solidária.

## Importância do Projeto Físico de Banco de Dados
- O projeto físico é a etapa de implementação prática do banco de dados, onde convertemos o planejamento (Fluxo e DER) para a realidade através da criação das tabelas, entidades e atributos no sistema. No contexto do nosso projeto, é nessa fase que definimos como serão armazenadas as informações de doadores e receptores, as imagens dos materiais e os locais cadastrados, além das permissões de acesso para os administradores.
- Para quem está aprendendo a programar, entender o projeto físico é importante porque permite visualizar como as informações funcionam "por trás" de cada sistema. Ao compreender como os dados são organizados e guardados, o estudante percebe a importância de construir estruturas seguras e eficientes, garantindo que o site seja confiável para todos os usuários que navegarem pela plataforma.

## Wireframe

- **O que é um wireframe?** 
O wireframe é como um guia visual que representa a estrutura esquelética de uma página web ou aplicativo. Ele funciona como um rascunho da interface, focando na organização da informação, distribuição dos elementos e navegação, sem se preocupar com aspectos estéticos (cores, imagens, tipografia final, etc).

- **Como criar um wireframe passo a passo:**
  - Entender o que a tela precisa fazer.
  - Desenhar ideias de layout à mão de como quer que o site seja.
  - Utilizar ferramentas digitais (como Figma, Miro ou Balsamiq).
  - Conectar as telas para validar se a navegação faz sentido lógico.
  - Ajustar com base em feedback antes de iniciar o design visual de alta fidelidade.

- **Importância do Sitemap**
  - O sitemap é fundamental porque mapeia todas as rotas do sistema. Ou seja, é a lista de todas as telas que precisa fazer e como elas se ligam.

- **Qual a importância do projeto de interface em nossos sites e aplicativos?**
  - Quando um sistema é intuitivo, conseguimos utilizar com facilidade, rapidez e sem frustrações. Manter padrões visuais e funcionais ajuda a preservar a familiaridade (que já conhecíamos), permitindo que o usuário continue desempenhando suas tarefas sem interrupções ou necessidade de reaprender tudo do zero.
  
  Sabemos que existem diversas pessoas com alguma dificuldade, seja visual ou alguém que não sabe mexer com tecnologia - como idosos. Por isso, tudo deve ser visado diante dessas situações ao criar um sistema. E tentar pensar como eles / se colocar no lugar deles, como eu gostaria que fosse. Todos queremos ser úteis, o sistema pode facilitar que eu possa fazer qualquer coisa da minha casa, portanto, que eu consiga utilizá-lo.

## Funcionalidades da Plataforma
O ambiente digital foi desenvolvido para ser intuitivo e seguro. Os principais módulos incluem:

* **Cadastro:** Registro obrigatório para validação de perfil.
* **Doação:** Catalogação de itens com descrição de estado de conservação.
* **Recepção:** Navegação por categorias e solicitação de retirada.
* **Avaliação:** Sistema de feedback para garantir a confiança da rede.

## Stack Tecnológica e Ferramentas

Para o desenvolvimento do EducaRecicla, foram selecionadas tecnologias que garantem escalabilidade, segurança e uma gestão eficiente de dados.
